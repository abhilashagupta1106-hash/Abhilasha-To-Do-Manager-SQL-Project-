# Abhilasha To-Do Manager (SQL Project)

## 📌 Description
This is a simple To-Do Management System built using SQL.  
I created this project to practice basic database operations like INSERT, UPDATE, DELETE, and SELECT.

## 🚀 Features
- Add new tasks
- View all tasks
- Update task status (Pending / Completed)
- Delete tasks

## 🛠️ Technologies Used
- SQL (MySQL)

## 📂 Database Structure
Table Name: tasks
- id (Primary Key)
- task_name
- status
- created_at

## ▶️ How to Run
1. Open MySQL
2. Run the SQL file (todo.sql)
3. Use SELECT queries to view tasks
